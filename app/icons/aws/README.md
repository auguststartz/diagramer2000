# AWS Icon Assets

Place official AWS Architecture PNG icons in this folder with these exact names:

- `ec2.png`
- `rds.png`
- `fsx.png`

You can also keep AWS's original filenames/folder structure under this directory. The renderer recursively scans for files containing `ec2`, `rds`, or `fsx` in the filename.

Use the AWS Architecture Icons pack from AWS, and export/copy the PNG variants for:

- Amazon EC2
- Amazon RDS
- Amazon FSx

The renderer will automatically use these files. If an icon is missing, a fallback placeholder shape is drawn.
