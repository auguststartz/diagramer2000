# Cloud Service Icons

Place PNG icon files in this directory for Cloud Services.

## Expected filenames

| Key              | Filename            |
|------------------|---------------------|
| office365        | `office365.png`     |
| hosted_epic      | `hosted_epic.png`   |
| entra            | `entra.png`         |
| okta             | `okta.png`          |

Icons should be square PNGs (recommended 128x128 or larger). They will be scaled to 36x36 inside the node.

If a PNG is missing, the renderer falls back to a colored rectangle with a 3-letter abbreviation.

## Sources

- `office365.png`: Microsoft 365 logo from Wikimedia (`Special:FilePath/Microsoft_365_logo.png`)
- `hosted_epic.png`: EPIC logo from `media.epic.com/epicdotcom/site/images/header/site-logo.png`
- `entra.png`: Microsoft Entra ID color icon from Wikimedia (`Special:FilePath/Microsoft_Entra_ID_color_icon.svg?width=512`)
- `okta.png`: Okta logo from Wikimedia (`Special:FilePath/Okta_logo_(2023).svg?width=512`)
