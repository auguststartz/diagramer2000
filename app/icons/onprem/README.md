# On-Prem Icons

Place PNG icon files in this directory for on-premises / Customer Network services.

## Expected filenames

| Key              | Filename          |
|------------------|-------------------|
| epic             | `epic.png`        |
| mfp              | `mfp.png`         |
| smtp             | `smtp.png`        |
| exchange         | `exchange.png`    |
| directory        | `directory.png`   |
| autoprint        | `autoprint.png`   |
| otfaim           | `otfaim.png`      |

Icons should be square PNGs (recommended 128x128 or larger). They will be scaled to 36x36 inside the node.

If a PNG is missing, the renderer falls back to a colored rectangle with a 3-letter abbreviation.
